## Logging Patch // No Admin

Logging commands such as 

- `logserver` 
- `logignore` 
- `logevents`
- `log` 

can function without Server Administrator (Ability) Permission and only require bot's Owner.

File changed:
 - `NadekoBot/NadekoBot.Core/Modules/Administration/LogCommands.cs`
